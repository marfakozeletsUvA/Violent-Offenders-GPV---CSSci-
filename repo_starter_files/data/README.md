# Data Directory

This folder contains project data.

## Structure

- `raw/` — Original, unmodified datasets (NOT tracked in git — add to .gitignore)
- `processed/` — Cleaned and merged data ready for analysis

## Data Not Included

Raw data files are not included in this repository due to size/licensing. 
See [DATA_SOURCES.md](../docs/DATA_SOURCES.md) for download links.
